Install ArchLinux

This is just a note, from the official ArchLinux Guide

  1. Test the Internet conn

    • ping archliux.org

    • Q:If failed to conn to the Internet
    • A:run this command twice

      systemctl enable dhcpcd.service

  2. Update System time

    • timedatectl status

    • timedatectl set-ntp true

  3. Start to Partition disks
    • see the disk status now

      • fdisk -l

      • find the partition you want to install ArchLinux and create the /part
    • Start to create partitions

      • cfdisk /dev/nvme or /dev/sda

      • Suggest to create a /home and a / for your system. If your pc donâ€™t have enough memory, remember to create the Swap partition.
    • Format the partitions

      • mkfs.ext4 /dev/..

      • mkswap /dev/swap partition -L Swap

      • swapon /dev/swap partition

  4. Mount your / partition
    • Mount the root partition /

    • mount /dev/nvme or sda /mnt

    • Mount the /home partition

    • mkdir /mnt/home

    • mount /dev/home partition /mnt/home

    • Mount the EFI partition

    • mkdir -p /mnt/boot/efi

    • mount /dev/efi partition /mnt/boot/efi

  5. Installation
    • If your local Internet not so good, consider to use a local mirror.
    • Write the Mirror link into â€˜/mnt/etc/pacman.d/mirrorlistâ€™

    • vi /etc/pacman.d/mirrorlist

    • Start to Install

    • pacstrap /mnt base base-devel linux linux-firmware vim

    • Configure the fstab & check whether the fstab is generated correctly

    • genfstab -U /mnt >> /mnt/etc/fstab

    • vi /mnt/etc/fstab *If correct generated, the partition information would exist in this file.

  6. Initialize the new system environment

    • arch-chroot /mnt

    • set timezone

    • ln -sf /usr/share/zoneinfo/Region/City /etc/localtime

    • hwclock â€“systohc

    • Localization

    • vi /etc/locale.gen ***remove the # front of ***

    • locale-gen

    • locale -a

    • echo LANG=en_US.UTF-8 > /etc/locale.conf

    • locale

    • Set hostname

    • echo myhostname > /etc/hostname

    • Configure the hosts
    • Set root password

    • passwd

    • Install some packages

    • pacman -S vim ntfs-3g networkmanager

    • Install CPU ucode
      • Intel CPU

      • pacman -S intel-ucode

      • AMD CPu

      • pacman -S amd-ucode

    • Install & Configure grub

    • pacman -S grub efibootmgr os-prober

    • grub-install â€“target=x86_64-efi â€“efi-directory=/boot/efi â€“bootloader-id=myLinux

    • grub-mkconfig -o /boot/grub/grub.cfg

    • Finally

    • systemctl enable dhcpcd.service twice

  7. Exit and Reboot

    • exit

    • umount -R /mnt

    • reboot

  8. Prepare for the first using

    • pacman -Syu

    • pacman -S bash-completion

    • Create a User

    • useradd -m -g users -G audio,video,network,wheel,storage -s /bin/bash yourname

    • passwd yourname

    • EDITOR=vim visudo remove # front of %wheel ALL=(ALL) ALL

    • exit

    • Login as the new user just created

    • sudo pacman -Syu

    • Install xorg xfce4-terminal

    • sudo pacman -S xorg-server xorg-apps xorg-xinit xterm xfce4-terminal

    • Install GPU drivers

    • lspci|grep -e VGA -e 3D

    • check which Graphic driver need to install. link

    • sudo pacman -S nvidia

    • Install cable net and wifi

    • sudo pacman -S wpa_supplicant dhcpcd dialog ppp net-tools netctl

    • Install xmonad window manager

    • sudo paxman -S xmonad xmonad-contrib xmobar stalonetray dmenu

    • vim ~/.xinitrc

    • Configure xmonad wm

    • xmonad â€“recompile

    • Install fonts

    • sudo pacman -S wqy-microhei ttf-arphic-ukai ttf-arphic-uming
